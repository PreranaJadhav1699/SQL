# SQL Basics

This repository contains basic SQL commands with syntax, descriptions, and examples for learning and practice.

The examples are written using a sample table named `placeofinterest` to demonstrate how SQL queries work.

---

## Commands Covered

The following SQL commands are included in this repository:

- SELECT  
- WHERE  
- COUNT  
- DISTINCT  
- LIMIT  
- INSERT  
- UPDATE  
- DELETE  

Each command includes:
- Syntax  
- Description  
- Example query  

All queries are written and stored inside the SQL file.

---

```
Basics/
 └── basics of SQL.sql
```

## Sample Table Used

All examples assume the following table exists:

```
placeofinterest
```

Example columns used:

- name  
- type  
- city  
- country  
- airport  

---

## How to Use

1. Open the file `basics of SQL.sql`.
2. Read the comments to understand the syntax and purpose of each command.
3. Run the queries in any SQL-supported database system.
4. Modify values and experiment with your own examples to practice.

---

## Purpose

This repository is created to:
- Learn core SQL commands
- Understand how queries work
- Practice database operations
- Build a reference for future use
